# README

Collection of algorithm problems and solutions.

## Setup

- install ruby (2.4.1p111)
- install bundler gem
- install elixir (1.8.1 (compiled with Erlang/OTP 21))
- install node (v8.11.4)
- run `./install.sh`

## Linter and formatting

run `./run_linter.sh`
